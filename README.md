# Aventura em Marte

## Description
Aventura em Marte is an exciting space exploration adventure game where players embark on a journey to uncover the secrets of Mars. With challenges ranging from resource management to extraterrestrial encounters, players must navigate the red planet's terrain and make critical decisions that will determine their survival and success.

## Features
- Engaging storyline with multiple endings
- Resource management simulation
- Unique alien encounters
- Interactive terrain exploration
- Stunning graphics and immersive sound effects

## Installation Instructions
To install Aventura em Marte, follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/username/aventura-em-marte.git
   ```
2. Navigate into the directory:
   ```bash
   cd aventura-em-marte
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```
4. Start the game:
   ```bash
   npm start
   ```

## Usage Instructions
Once the game is running, follow the on-screen instructions to navigate the game world. Make choices that affect the storyline and manage your resources wisely to succeed in your mission.
